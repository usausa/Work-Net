# EnergyLineHmiIso

WPF + SkiaSharp による **工場エネルギーライン監視 HMI** の **クォータービュー（2:1 アイソメトリック）** 版サンプルです。
[EnergyLineHmi](../EnergyLineHmi/README.md) と同じプラントを、立体的な近未来風グリッドビューで俯瞰します。

![EnergyLineHmiIso のスクリーンショット](screenshot.png)

## 概要

- 敷地グリッド 16×13（z は高さ）のワールド座標を 2:1 投影で仮想キャンバス 1600×900 に描画します。
- エネルギーの流れを、流量に比例した速度で配管上を移動する **光のパルス** として表現します。
- シミュレーションロジック（`PlantSimulation`）は EnergyLineHmi と同等で、見せ方（レンダラ）だけを差し替えた構成です。

## 表示内容

| 区分 | 内容 |
|---|---|
| 構造物 | 受電変電所・主変圧器・太陽光パネル・ガスエンジン CGS・冷却塔・貫流ボイラ・生産ライン A/B・ユーティリティ・乾燥炉・給水/補給水タンクを立体ボックスで配置 |
| エネルギーフロー | 電力・都市ガス・蒸気・復水/給水・冷却水を色分けし、グロー付きの光点が流量に応じて流れる。分岐ノードは発光ポイントで表示 |
| HUD パネル | KPI サマリ、受電デマンドリング、各種ミニ計器、トレンド、イベント履歴 |
| 演出 | 敷地外周のグロー、四隅のブラケット、走査ライン、異常機器のアラームリング |

## 操作

| 操作 | 動作 |
|---|---|
| 構造物をクリック | 起動 / 停止を切替（手前の建物を優先するヒットテスト） |
| ホバー | フットプリントを発光させて強調 |

## 実行方法

```powershell
dotnet run -c Release
```

> 要 .NET 8 SDK / Windows（WPF）。

## 構成

```
MainWindow.xaml(.cs)         SKElement の配置とマウス操作・再描画ループ
Rendering/IsoRenderer.cs     2:1 投影・背景グリッド・エネルギーフロー描画
Rendering/IsoBuildings.cs    建物・タンク・冷却塔・ソーラーなど構造物のアイソメ描画
Rendering/IsoHud.cs          ヘッダ・KPI・デマンドリング・計器・トレンド・イベント
Rendering/IsoTheme.cs        配色・グロー（MaskFilter）・フォント定義
Simulation/PlantSimulation.cs  各計測値のシミュレーションと警報ロジック
```
