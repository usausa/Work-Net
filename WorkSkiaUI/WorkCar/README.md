# WorkCar — TYPE-19 STRATOS Cockpit Telemetry

WPF + SkiaSharp による **近未来レーシングカーのコックピット HUD** デモです。
車両シミュレータが走行シーンを自動再生し、タコメータや各種計器がリアルタイムに動きます。

![WorkCar のスクリーンショット](screenshot.png)

## 概要

- 仮想座標 1920×1080 にレイアウトし、ウィンドウサイズへ等比スケールして描画します。
- `VehicleSimulator` が **ストレート → ブレーキ → コーナー** のフェーズを自動で繰り返し、計器値を生成します。
- スキャンライン・走査帯・パースペクティブグリッドなど、CRT 風の演出を全面に施しています。

## 表示内容

| 区分 | 内容 |
|---|---|
| 上部バー | LAP（周回） / TIME / BEST / POS（順位） / VMAX（最高速） / SYS、タイトル |
| タコメータ | 〜19,000rpm の大型ゲージ、レッドライン、シフトライト |
| 中央クラスタ | スピード（km/h）、ギア表示（1〜8）、ブーストポット残量 |
| ダイヤル・ゲージ | ERS 出力・G などのサイドダイヤル、下部に 水温 / 油温 / 油圧 / 燃料 / バッテリー / ターボ圧 |
| 警告 | 水温・油温・燃料残量・ブースト枯渇を検知して点滅表示 |
| フッタ | 擬似テレメトリストリーム、リンク強度、操作ヘルプ |

## 操作

| キー | 動作 |
|---|---|
| `Space` | ブースト（押している間、充電が続く限り発動） |
| `P` | 一時停止 / 再開（TELEMETRY HOLD） |
| `Esc` | 終了 |

## シミュレーション

ギア比・スロットル / ブレーキ・ターボ過給・ERS 回生・縦横 G・水温 / 油温 / 油圧 / 電圧 / 燃料を物理風のモデルで生成します。
燃料が尽きると自動的に「ピット給油」で満タンに戻ります。

## 実行方法

```powershell
dotnet run -c Release
```

> 要 .NET 10 SDK / Windows（WPF）。

## 構成

```
MainWindow.xaml(.cs)       SKElement の配置・フレームループ・キー入力
HudRenderer.cs             HUD 全体のレイアウトと背景 / 上部バー / 警告 / フッタ
HudRenderer.Gauges.cs      タコメータ・スピード・ギア・ダイヤル・各ゲージ
HudParts.cs                グローテキスト・カットパネルなどの描画パーツ
VehicleSimulator.cs        走行フェーズと計器値のシミュレーション
```
