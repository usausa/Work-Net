# MechHud — WPF + SkiaSharp 陸戦ロボットHUDデモ

FighterHud をベースにした、地上戦主体の人型機動兵器(メカ)向け戦術HUDです。
グリーンフォスファー調の配色で、疑似シミュレーション(`MechSim`)がすべての表示を自動駆動します。

## 実行方法

```powershell
dotnet run --project MechHud.csproj
# または
dotnet build -c Release
.\bin\Release\net8.0-windows\MechHud.exe
```

必要環境: .NET 8 Desktop Runtime / Windows

## 操作

| キー | 動作 |
|------|------|
| `F11` | フルスクリーン切替 |
| `Esc` | 終了 |

## 戦闘機版との相違点(本機の特徴)

- **移動モード**: WALK → DASH → NOE FLT(跳躍ユニットによる匍匐飛行)を自動サイクル。
  歩行時は脚部の上下動(ボブ)、ホバー時はAGL(対地高度)表示が生きる
- **推進剤残量**: 左右跳躍ユニット個別のセグメントゲージ(PL/PR)。
  ホバー/ダッシュで消費、歩行中に回復。25%未満で警告
- **ダメージ状況**: 機体シルエット8部位(HEAD/CORE/L-R ARM/L-R LEG/L-R JU)を
  HP色分け表示(緑→琥珀→赤点滅)。被弾時は IMPACT 警告+部位フラッシュ、緩速自動修復
- **左右武装切り替え**: L-ARM(サブマシンガン)⇔ R-ARM(長距離砲/ミサイルボックス交互)を
  12秒周期でスワップ。アクティブ側パネル強調+「WPN SWAP」表示+照準レチクルも武装別に変化
  - 残弾は数値表示に加え、**武装シルエット内のゲージ**でも表示
    (SMG=マガジン充填、長距離砲=砲弾ラック12セル、ミサイル=発射管8セル)
  - リロード進捗バー、チャージ(CHG)バー、発射エフェクト付き
- **通信パネル**: CH1 オープン回線(周波数+回線ID)/ CH2 秘匿回線(ENC・鍵IDが定期更新)/
  CH3 指揮回線。アクティブ回線ハイライト、送受信波形、発信元コールサイン表示
- **レーダー+地形表示**: ヘディングアップレーダー(3km)に加え、等高線付き戦術マップ
  (マーチングスクエア法で生成、グリッド座標 A-F/1-4)
- **部隊編成**: 中隊 DAGGER 12機 = 小隊 D1/D2/D3 各4機。
  自機小隊(D1)は各機の部隊コード付きで、他小隊はクラスタ+小隊コードでマップに表示。
  敵性コンタクト(E1-E5)・目標地点(OBJ-A)もマップに反映

## 構成

| ファイル | 内容 |
|----------|------|
| `MechSim.cs` | 移動モード・推進剤・部位ダメージ・武装・通信・コンタクト・部隊位置の疑似シム |
| `MechRenderer.cs` | SkiaSharpによる全描画(等高線マップはキャッシュ生成) |
| `MainWindow.xaml(.cs)` | `SKElement` + `CompositionTarget.Rendering` 描画ループ |

## カスタマイズの起点

- 配色: `MechRenderer` 先頭の静的カラー(`Main` を変えると全体の雰囲気が変わります)
- 部隊編成・コールサイン: `MechSim` コンストラクタの `SquadUnit` リスト
- 地形: `MechRenderer.Height()` のサイン波合成と `levels` 配列
- 武装サイクル・ダメージ頻度: `MechSim.Update()` の各タイマー定数
